# Проект интернет-магазина

## Описание
Интернет магазин написан на языке программирования Python с помощью фреймворка Django.
Фреймворк Django - это набор инструментов, библиотек и правил, которые помогают разработчикам создавать приложения, упрощая и ускоряя процесс разработки. Фреймворк предоставляет структуру, внутри которой можно разрабатывать веб-приложения.
Django использует шаблон проектирования MTV (Model — Template — View).

## Описание Контроллеров
Приложение Catalog - модуль контроллеров view.py:
1. Класс ProductListView - Контроллер для отображения списка товаров на странице home.html
2. Класс ContactListView - Контроллер для отображения страницы contacts.html и обработки POST запроса.
3. Класс ProductDetailView - Контроллер для отображения страницы с подробным описанием товара.
4. Класс ProductCreateView - Контроллер для добавления товаров в базу данных.

Приложение Blogs - модуль контроллеров view.py:
1. Класс BlogListView - Контроллер для отображения списка статей.
2. Класс BlogCreateView - Контроллер для создания статьи.
3. Класс BlogDetailView - Контроллер для отображения страницы с подробным описанием статьи.
4. Класс BlogUpdateView - Контроллер для изменения статьи.
5. Класс BlogDeleteView - Контроллер для удаления статьи.

## Описание Страниц
В приложении Catalog папка templates содержит следующие веб страницы:
1. base.html - Базовый шаблон
2. home.html - Главная станица сайта
3. contacts.html - Страница с контактами и формой обратной связи.
4. product.html - Страница с описанием товара.
5. add_product - Страница для добавления товаров.

В приложении Blogs папка templates содержит следующие веб страницы:
1. blog_list.html - Страница списка статей.
2. blog_create.html - Страница для добавления статьи.
3. blog_detail.html - Страница с описанием статьи.
4. blog_confirm_delete.html - Страница подтверждения удаления статьи.


## Описание Моделей
Приложение Catalog - модуль моделей models.py:
1. Category - Описание полей модель категорий товаров
2. Product - Описание полей модель товаров
3. Contact - Описание полей контактных данных

Приложение Blogs - модуль моделей models.py:
1. Blog - Описание полей модели статей.

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/NasgDen/Django
```
2. В PyCharm установите менеджер проекта poetry.
3. В PyCharm устанавливаем все зависимости проекта из файла pyproject.toml
4. Переименуйте файл .env.sample в .env. Заполните файл конфигурации .env параметрами для подключения к базе данных.

## Использование:

1. В терминале введите: python3 manage.py runserver
2. Сервер запускается по адресу: 127.0.0.1:8000
3. В браузере введите адрес: http://127.0.0.1:8000/
